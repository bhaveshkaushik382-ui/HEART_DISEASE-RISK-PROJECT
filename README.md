## 📌 Project Overview

AI Health Risk Dashboard is a web application that predicts a user's health risk based on multiple input parameters.

The system analyzes user-provided information using a trained Machine Learning model and generates easy-to-understand recommendations using an AI API.

This project combines Machine Learning and Generative AI to transform raw health data into meaningful insights.
